## Features
- Multistep booking process
- Stripe payment gateway integration
- User-friendly interface

## Requirements
- WordPress 5.0 or higher
- PHP 7.0 or higher
- Composer

## Installation


```bash
git clone https://github.com/Tapabrata-Goswami/multistep-booking.git

cd multistep-booking
```
